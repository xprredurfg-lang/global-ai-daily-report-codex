# AGENTS.md｜全球AI每日动态报告项目规则

## 执行原则
以事实依据为荣，以凭空编造为耻。
以问题定义为荣，以盲目开写为耻。
以逻辑推演为荣，以结论跳跃为耻。
以场景落地为荣，以空泛套话为耻。
以边界清晰为荣，以越权幻想为耻。
以结构化输出为荣，以散乱堆料为耻。
以自查验证为荣，以不检即交为耻。
以简洁高密为荣，以冗长废话为耻。

## 执行链
读取 daily_prompt.md → configs → templates → 联网研究 → 筛选 → 撰写 → 图片生成 → docx交付。

## 图片规则
Codex直接调用可用图片生成能力。
图片必须基于已核验事实。
先形成内部visual_brief，再生成图片。
图片必须直接嵌入Word报告。
不输出图片提示词文件。

## 交付规则
最终交付：
reports/YYYY-MM/全球AI每日动态报告_YYYY-MM-DD_制造质量部门专用版.docx

图片失败不阻断正文交付，只记录降级状态。

## 失败规则
docx无法生成、正文为空、无法保存时生成failure_report。
